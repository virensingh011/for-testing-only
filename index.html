Graph Risk Propagation Engine

                one deeper technical subsystem
              

              

                

                  Propagation equation
                  
R_i(t+1) = alpha H_i + beta E_i + gamma N_i(t) + delta C_i - lambda P_i

                  
Nodes represent regions. Edges represent transportation, communication, or infrastructure links. Neighbor risk flows through weighted edges.

                

                

                  Monte Carlo idea
                  
Each run adds uncertainty to hazard, exposure, preparedness, and edge flow. The app estimates average final risk and spread range.

Why I built this
Designed as an independent STEM exploration project while experimenting with browser-based simulations, maps, data retrieval, and animated models.

                
Learning goals
Learn client-side simulation systems, AI-style visualization, interactive science modeling, vector search basics, and frontend architecture.

                
Limitations
This runs entirely in the browser without a backend. Disaster, science, and threat data are simulated for visualization and research practice.
